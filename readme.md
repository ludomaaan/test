# Эльфы против Дракона

Простой 2D-платформер на чистом HTML5 Canvas. Управляйте маленьким эльфом, собирайте магические кристаллы, спасайте друзей и сразитесь с драконом в финальном уровне.

## Запуск
1. Откройте `index.html` в любом современном браузере (Chrome, Firefox, Edge).
2. Если браузер блокирует локальные файлы, запустите простой статический сервер: в терминале в папке проекта выполните `python -m http.server 8000` и откройте страницу `http://localhost:8000`.
3. Управление: ← → — движение, пробел — прыжок, `X` — атака против дракона.
4. Прогресс (максимальный достигнутый уровень и лучший счёт по кристаллам) сохраняется в `localStorage`. Кнопка «Продолжить» становится активной, когда есть сохранённые данные; «Начать заново» обнуляет прогресс.

## Особенности
- 4 уровня: Волшебный лес, глубокий лес, горы циклопов и логово дракона.
- Враги: волки, циклопы и финальный босс-дракон с огненными шарами.
- Подвижные платформы, ямы, портал на уровень и спасаемые эльфы.
- Очки за кристаллы и дополнительные жизни от союзников.
